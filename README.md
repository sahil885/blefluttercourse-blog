# BLE Flutter Course — Blog

Organic SEO blog for [blefluttercourse.com](https://blefluttercourse.com).

Built with Next.js 14 + Tailwind CSS, deployed on Vercel.

## Adding a new blog post

1. Create a new `.md` file in `content/posts/`
2. Add frontmatter at the top:

```md
---
title: "Your Post Title"
date: "2025-06-01"
excerpt: "A one or two sentence summary shown in post cards and meta tags."
tags: ["Tag1", "Tag2"]
---

Your content here in Markdown...
```

3. Commit and push — Vercel auto-deploys.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
